🚨 AI Emergency Assistant

An AI-powered web app that provides quick first-aid instructions during emergencies like fire, injuries, and medical situations.

---

# Features

- Emergency guidance (Fire, Injury, Heart cases)
- Multi-language support (English, Tamil, Hindi)
- Voice output using gTTS
- Simple Streamlit interface

---

# How to Run

pip install -r requirements.txt
streamlit run app.py

---

# API Key Setup

Windows

set OPENROUTER_API_KEY=your_api_key_here
